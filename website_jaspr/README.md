# website_jaspr

A Jaspr-based migration workspace for the `llamadart` docs site.

## Running the project

Run your project using `jaspr serve`.

The development server will be available on `http://localhost:8080`.

## Building the project

Build your project using `jaspr build`.

The output will be located inside the `build/jaspr/` directory.

## Migration intent

- Keep markdown as canonical content format.
- Rebuild docs navigation/layout in Jaspr.
- Preserve release-tag-based docs deployment flow after cutover.
